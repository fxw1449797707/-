# 🌍 世界地图应用

一个美观且功能丰富的交互式世界地图应用，使用 Leaflet.js 构建。

## ✨ 功能特色

- 🗺️ **多种地图样式**: 标准地图、卫星图、地形图、暗色主题
- 🌏 **大洲视图**: 快速跳转到世界各大洲
- 📍 **城市标记**: 显示世界主要城市及其信息
- 🎯 **点击定位**: 点击地图任意位置查看坐标
- ⌨️ **键盘快捷键**: 便捷的键盘操作
- 💡 **地理知识**: 自动显示有趣的地理小知识
- 📱 **响应式设计**: 适配各种屏幕尺寸

## 🚀 使用方法

1. **打开地图**: 直接在浏览器中打开 `index.html` 文件
2. **交互操作**:
   - 鼠标滚轮缩放地图
   - 拖拽移动地图视图
   - 点击地图查看坐标
   - 点击城市标记查看详细信息
3. **控制面板**: 使用右侧面板切换地图样式和视图
4. **键盘快捷键**:
   - `R`: 重置到世界视图
   - `M`: 切换城市标记显示/隐藏
   - `1-7`: 快速跳转到各大洲

- **Leaflet.js**: 开源地图库

- **响应式设计**: 适配移动设备
